## How to Use
1. **Prepare:** Park Michael in the Grand Senora Desert.
2. **View:** Switch to **First Person View** (Default key: `V`) for the most efficient running circle.
3. **Launch:** Run `KifflomRunner.exe` as Administrator.
4. **Activate:** Press `CTRL + SHIFT + ENTER` to start the run.
5. **Monitor:** Use a second monitor to watch the timer.
6. **Stop:** Alt-Tab to the app window and press `ESC` to release all keys and exit.
